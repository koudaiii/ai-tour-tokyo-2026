```sh

```

```sh

```

Create

```sh
script/launch-vm
```

List

```sh
script/list-resource-groups
```

Destroy

```sh
script/destroy-vm
```

## Script Usage

- `script/bootstrap`: Prepare local development environment (dependency, DB, and memcached checks included)
- `script/server`: Start the web application
- `script/restore`: Restore PostgreSQL dump
- `script/backup`: Create PostgreSQL dump
- `script/upload-to-github`: Upload dump file to GitHub Releases
- `script/launch-vm`: Create Azure VM
- `script/list-resource-groups`: List Resource Groups tagged for this repository
- `script/destroy-vm`: Delete Resource Group(s)

## Make Commands

- `make init`: Download PostgreSQL dump and benchmarker image fixtures
- `make restore`: Restore `sql/isuconp_data.dump` into PostgreSQL
- `make clean`: Remove local artifacts downloaded/generated by `make init`
